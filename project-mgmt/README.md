# Project Management

This folder contains the initial planning artifacts for the Apeirograph local harmonic intelligence lab.

## Structure

- `epic-local-harmonic-intelligence-lab-mvp.md` — the overarching MVP objective
- `sprint-01-foundation-and-proof-of-value.md` — the first sprint plan
- `issues/` — bite-sized implementation issues with measurable acceptance criteria

## Working principle

The project should move in small, testable increments. Each issue should end with a verifiable outcome and should avoid premature infrastructure work.
