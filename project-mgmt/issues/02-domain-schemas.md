# Issue 02: Domain Schemas

## Goal

Define the core Pydantic schemas for harmonic objects and structured responses.

## Why it matters

Reliable schemas create a stable contract between prompts, model outputs, and evaluation tools.

## Tasks

- Define chord, scale context, and progression input models.
- Define explanation and suggestion response models.
- Add a few representative JSON examples.
- Validate that sample objects load correctly.

## Definition of done

- Core schema files exist.
- Sample inputs validate successfully.
- Structured outputs are clearly defined for downstream use.

## Out of scope

- Large musical corpora
- Full transform engine implementation

## Estimate

Small
