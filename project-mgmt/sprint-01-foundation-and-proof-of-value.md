# Sprint 01: Foundation and Proof of Value

## Sprint goal

Establish the smallest serious prototype of the local harmonic intelligence lab and verify that it provides useful, constrained reasoning in the Apeirograph domain.

## Included issues

1. Repository scaffold
2. Domain schemas
3. Ollama client integration
4. Explain workflow MVP
5. Suggest workflow MVP
6. Eval starter pack

## Sprint definition of done

- The project can run locally.
- One local model is configured and callable.
- Explanation and suggestion flows work on sample inputs.
- At least five curated evaluation cases exist.
- Results can be reviewed for correctness, clarity, and usefulness.

## Risks to watch

- Scope creep into UI polish or infrastructure
- Overly broad prompts that drift into generic music assistant behavior
- Lack of measurable eval criteria

## Recommended order

Start with issues 1 through 3, then build 4 and 5, then close the sprint with issue 6.
